# Mock Twitter App

This is Mock Twitter App.

## Stack
* `react`
* `redux`
* `react-redux` — binding of Redux for React.
* `react-dom`

For CSS UI i have used Material UI which provides tons of react components
* `material-ui/core`

* `redux-thunk` —  compose actions for Redux.
* `react-router-dom` — routing for React.


### Login page
User should enter login name

### Timeline
After Login it will show "what is happening" form and default tweet

### Reply to tweet
There is functionilty where you can reply the tweet.
